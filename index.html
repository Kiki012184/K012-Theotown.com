Perkenalan

Hai saya kiki012 dari jawa timur - Indonesia,dan saya akan memperkenalkan diri kepada anda.

Saya adalah sebuah programmer autodidak dan suka bermain video game,dan berumur 15 tahun,anda bisa memanggil saya sebagai kiki jika anda mau,jadi saya membuat banyak plugin di game TheoTown, kebanyakan bertemakan transportasi,situs ini saya buat untuk para kreator lain yang memberi credit kepada saya.

Aku membuat beberapa konten pada beberapa sosial media, contohnya YouTube,saya membuat video gameplay beberapa game,akan tetapi channel YouTube saya terlalu sepi,dan mengunggah konten acak.
Di Facebook,dan Instagram aku membuat postingan tentang spoiler pembaruan plugin theotown saya, berisikan foto-foto dan terkadang video spoiler tersebut

Aku memiliki akun sosial media lain,akan tetapi jarang digunakan,jadi hanyalah beberapa sosial media saja yang sering aktif,dalam hal politik saya menyukai negara" eks-uni Soviet, eks-Yugoslavia, Negara-negara Pakta warsaw, China (PRC),dan Korea (DPRK).
